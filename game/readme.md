# 游戏名：秀了又秀

## 游戏简介
欢迎来到《秀了又秀》，这是一款充满乐趣和挑战的策略消除游戏。

## 游戏玩法
- **消除挑战**：将方块点击使其位于下方方框，当出现三个相同种类的方块即可消除，消除越多，得分越高；但是要注意哦，下方最多只能存在7个方块。

## 游戏特色
- **精美的画面**：游戏采用可爱的动漫风格，冰雪风景。
- **简单的操作**：只需简单的点击，即可享受游戏的乐趣。
- **社交互动**：与朋友一起竞赛，看看谁能够更快更好地完成关卡。
